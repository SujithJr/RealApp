# Tracker App
##### Web app to track time of projects, add members, create workspace
###### (Under Development) 

Front End: `NuxtJS`
Back End: `NodeJS` and `MongoDB`

##### Client
After cloning the repo, run:
```
$ npm install
$ npm run dev
```

##### Server
After cloning the repo, run:
```
$ npm install
$ npm start
```
